Quotests

2015-07-07 · Λογισμικό Γλώσσες · android αγγλικά τεστ

This application consists of multiple choice quizzes to test your English language skills. But there is a twist. Instead of using common boring phrases, it features text from great quotes by famous people!

Αυτή η εφαρμογή περιέχει τεστάκια με ερωτήσεις πολλαπλής επιλογής για να ελέγξετε τις ικανότητές σας στην αγγλική γλώσσα. Αλλά υπάρχει μια σημαντική ανατροπή. Αντί για ανιαρές τετριμένες φράσεις, σας παρουσιάζει καταπληκτικά γνωμικά από διάσημους ανθρώπους!
